# How to run this project

1. Install all dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Run all tests:
   ```bash
   npm test
   ```

4. Compile TypeScript to JavaScript:
   ```bash
   npx tsc
   ```
